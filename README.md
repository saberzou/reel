# Reel — A Cinema Journal

Films, reimagined as posters. A slow personal record by Saber Zou.

Live: https://saberzou.github.io/reel

## Stack
- Static HTML/CSS/JS — zero build step
- WebGL displacement shader on each poster (idle breathing + hover ripple)
- Justine Soulié–inspired noise grain layer
- Horizontal scroll filmstrip with snap

## Local
Open `index.html` in any modern browser. That's it.

## Deploy
Hosted via GitHub Pages from the `main` branch root.
